# Postman API Testing

Proyecto básico de API REST desarrollado con Node.js, Express y Postman.

## Tecnologías utilizadas
- Node.js
- Express
- Postman
- JSON

## Métodos HTTP implementados
- GET
- POST
- PATCH
- DELETE

## Testing
Pruebas de endpoints realizadas mediante Postman.

## Estructura del proyecto
- `app.js` → servidor principal
- `movies.json` → almacenamiento de datos
- `Movies.postman_collection.json` → colección de pruebas API

## Persistencia
Datos almacenados en archivo JSON local.
