# Postman API Testing

Proyecto básico de API REST utilizando Node.js, Express y Postman.

## Tecnologías
- Node.js
- Express
- Postman

## Métodos HTTP implementados
- GET
- POST
- PATCH
- DELETE
- Testing de endpoints

## Archivos
- app.js → servidor
- movies.json → datos
- postman collection → pruebas API

## Persistencia
Datos almacenados en archivo JSON
